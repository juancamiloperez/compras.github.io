API REST que devuelve JSON (requiere CORS habilitado)

Requisitos endpoint REST

1
Responder con Content-Type: application/json y array de objetos
2
El servidor debe tener CORS habilitado para el dominio que abre esta herramienta
3
Token Bearer opcional. Si la respuesta viene anidada, usa Ruta al array (ej: data.items)
Requiere que TI habilite CORS en los endpoints. Sin esto el navegador bloqueara las peticiones.
